## 📖 Overview

<a href="#monit">Monit</a> is a lightweight system resource monitor for Mac that displays CPU, memory, disk, and network statistics in the menu bar — providing always-visible system performance awareness without opening Activity Monitor or running a full-screen monitoring dashboard.

The <a href="#monit">menu bar resource display</a> shows current CPU usage, memory pressure, and other key metrics directly in the Mac menu bar — a persistent at-a-glance view of system resource consumption that updates in real time without requiring any window to be open. The specific metrics displayed in the menu bar are configurable, allowing the user to show only the information most relevant to their monitoring needs. The <a href="#monit">detailed popover on click</a> expands to show comprehensive system information when the menu bar icon is clicked — CPU usage per core, memory usage breakdown, disk read and write activity, network upload and download throughput, and battery status in a clean popover interface without launching a separate application window.

The <a href="#monit">process list by resource usage</a> shows the top CPU-consuming and memory-consuming processes at a glance — identifying which applications and processes are responsible for high CPU or memory usage without opening Activity Monitor and navigating to the CPU or Memory column sort. The <a href="#monit">network throughput monitoring</a> displays real-time upload and download speeds — useful for monitoring download progress, verifying that network activity is expected, and identifying unexpected background network usage during time-sensitive tasks.</p>

<a href="#monit">Memory pressure display</a> shows macOS memory pressure status — the system-level indicator of whether the Mac has adequate free memory or is under memory pressure that may be affecting performance. <a href="#monit">Disk activity monitoring</a> shows current disk read and write rates — identifying disk-intensive operations and monitoring storage system activity during backups, file transfers, and other storage-heavy tasks.

<a href="#monit">Battery information</a> displays battery charge level, charging status, time remaining, cycle count, and battery health information — the battery details that macOS's own battery menu provides plus additional health information. <a href="#monit">Lightweight footprint</a> — Monit is designed to consume minimal CPU and memory itself, providing monitoring without meaningfully contributing to the resource usage it monitors.</p>

Monit's value is in eliminating the friction of checking system resources. Opening Activity Monitor, waiting for it to load, finding the metric of interest, and closing it again takes fifteen to thirty seconds and interrupts workflow. CPU usage, memory pressure, and other configurable metrics — the specific metrics displayed in the menu bar are user-configurable.

Yes. The popover shows a list of top CPU-consuming and memory-consuming processes.

<details>

Yes. Real-time upload and download throughput is displayed in the popover and optionally in the menu bar.

Yes. Battery charge, charging status, cycle count, and health information are displayed.

</details>

Monit is designed for minimal resource consumption — it monitors system resources without meaningfully contributing to the CPU and memory usage it displays.

Yes. Disk read and write rates are displayed in the popover.

</details>
